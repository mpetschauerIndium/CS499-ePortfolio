Professional Introduction

Computer Science Portfolio

This ePortfolio presents my growth through the Southern New Hampshire University Computer Science program and my completed CS499 capstone. The portfolio centers on Travlr Getaways Admin, a full-stack administrative application enhanced with Angular, TypeScript, Express.js, Node.js, MongoDB, Mongoose, JWT authentication, protected routes, and secure password storage.

The project reflects the skills I want employers to see: clear software architecture, secure API design, database-backed CRUD workflows, technical communication, and the ability to improve an existing codebase into a more maintainable and scalable application.
